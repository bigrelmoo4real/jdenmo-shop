# JDENMO SHOP - E-commerce Platform

Bienvenue sur JDENMO SHOP ! Une plateforme e-commerce moderne pour vendre des vêtements en Côte d'Ivoire.

## 🚀 Déploiement GRATUIT en 3 minutes

### Option 1 : Render.com (Recommandé ⭐)

1. Créez un compte sur [render.com](https://render.com)
2. Git push ce projet sur GitHub
3. Dans Render, cliquez "New +" → "Web Service"
4. Sélectionnez votre repo GitHub "JDENMO SHOP"
5. Configure :
   - **Name:** jdenmo-shop
   - **Root Directory:** (vide)
   - **Build Command:** `npm install`
   - **Start Command:** `npm start`
6. Cliquez "Deploy" 🚀

Votre site sera accessible via : `https://jdenmo-shop.onrender.com`

---

### Option 2 : Railway.app (Aussi gratuit)

1. Allez sur [railway.app](https://railway.app)
2. Connectez votre GitHub
3. Importez ce repo
4. Railway détecte automatiquement que c'est Node.js
5. Laissez déployer
6. Obtenez une URL publique !

---

### Option 3 : Vercel (Pour API simple)

1. Allez sur [vercel.com](https://vercel.com)
2. Importez depuis GitHub
3. Cliquez "Deploy"
4. Obtenez une URL permanente

## 📋 Structure du projet

- `server.js` - Serveur Express (API + fichiers statiques)
- `index.html` - Page d'accueil
- `cart.html` - Panier d'achat
- `admin.html` - Panneau de gestion des produits
- `login.html` - Connexion admin
- `products.json` - Base de données des produits
- `uploads/` - Stockage des images produits
