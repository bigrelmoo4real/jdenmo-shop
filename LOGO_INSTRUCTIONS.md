# 📝 Instructions pour Ajouter Votre Logo JDENMO SHOP

## Comment intégrer votre logo image au site

Votre site est maintenant prêt à afficher un logo image ! Voici comment faire :

### Étape 1: Choisissez où placer votre logo
Vous pouvez mettre votre fichier logo dans l'un de ces emplacements :

1. **À la racine du projet** (recommandé)
   - Chemin: `C:\Users\HP\Desktop\JDENMO SHOP\logo.png`
   
2. **Dans un dossier assets**
   - Chemin: `C:\Users\HP\Desktop\JDENMO SHOP\assets\logo.png`
   
3. **Dans le dossier uploads**
   - Chemin: `C:\Users\HP\Desktop\JDENMO SHOP\uploads\logo.png`

4. **Dans un dossier images**
   - Chemin: `C:\Users\HP\Desktop\JDENMO SHOP\images\logo.png`

### Étape 2: Nommez votre fichier
- **Nom attendu:** `logo.png` (ou `.jpg`, `.jpeg`, `.svg`, `.gif`)
- Le site cherchera automatiquement ce fichier dans l'ordre des dossiers ci-dessus

### Étape 3: Vérifiez dans le navigateur
1. Mettez votre fichier logo dans l'un des dossiers
2. Rafraîchissez la page (F5)
3. Le logo apparaîtra automatiquement à la place du texte "JDENMO SHOP"

## 📐 Recommandations pour le logo

- **Dimensions:** 120px x 50px (maximum)
- **Format:** PNG (transparent) ou SVG idéal
- **Couleurs:** Compatible avec le vert (#27AE60) et orange (#FF6B35) du site
- **Poids:** Moins de 100KB pour performances optimales

## 🔄 Ce qui se passe automatiquement

Le site vérifie dans cet ordre :
1. `logo.png` à la racine
2. `assets/logo.png`
3. `uploads/logo.png`
4. `images/logo.png`

Si aucun logo n'est trouvé, le texte "JDENMO SHOP" s'affiche à la place.
